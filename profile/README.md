# Black Duck Integrations Workflow Samples                                                                                                                                                                                               
                                                                                                                                                                                                                                         
Welcome to the Black Duck CI/CD Integrations samples repository. It contains comprehensive collection of live, executable Black Duck Security Scan Plugin samples for Black Duck SCA, Coverity, and Polaris security products. Find ready-to-use integration samples to quickly implement security scanning in your CI/CD pipelines.                                        
                                                                                                                                                                                                                                         
## Workflow Samples                                                                                                                                                                                                                      
| Product | Details | Samples |                                                                                                                                                                                             
|---------|-------------|-------------------|                                                                                                                                                                                            
| **Black Duck SCA** | Analyzes third-party components and open-source dependencies to identify security vulnerabilities, license risks, and compliance issues | [Samples](https://github.com/blackducksca-jenkins-samples) |                                                                                                          
| **Coverity** | Static code analysis that examines source code for security vulnerabilities, coding defects, and quality issues without executing the application | [Samples](https://github.com/coverity-cnc-jenkins-samples) |                                                                                                         
| **Polaris** | Cloud-based platform combining static application security testing and software composition analysis for comprehensive code and dependency security scanning | [Samples](https://github.com/polaris-jenkins-samples) |
